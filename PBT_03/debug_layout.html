Cách 1: Sử dụng box-sizing: border-box

Width bây giờ BẰNG width + padding + border

Sidebar

Width: 300px

Padding: 20px

Border: 1px

Thực tế: 300px (đã bao gồm)

Content

Width: 660px

Padding: 30px

Border: 1px

Thực tế: 660px (đã bao gồm)

Total: 300px + 660px = 960px ✓

Cách 2: Không dùng box-sizing, giảm width để fit

Tính toán lại width để vừa với container

Sidebar

Width: 258px (300 - 42)

Padding: 20px

Border: 1px

Thực tế: 300px

Content

Width: 598px (660 - 62)

Padding: 30px

Border: 1px

Thực tế: 660px

Total: 300px + 660px = 960px ✓

Giải Thích Chi Tiết

Vấn đề Gốc:

Sidebar: 300 + 20 + 20 + 1 + 1 = 342px
Content: 660 + 30 + 30 + 1 + 1 = 752px
Tổng: 1094px > 960px (container) → Content bị đẩy xuống!

Cách 1 - border-box (Hiện đại):

box-sizing: border-box;
→ Width = width + padding + border
Sidebar: 300px (thực tế)
Content: 660px (thực tế)
Tổng: 960px ✓

Cách 2 - Tính toán manual:

Sidebar: 300px - (20+20+1+1) = 258px width
Content: 660px - (30+30+1+1) = 598px width
Thực tế:
  Sidebar: 258 + 42 = 300px
  Content: 598 + 62 = 660px
Tổng: 960px ✓
